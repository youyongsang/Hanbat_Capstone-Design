# 주제

경량 딥러닝 모델을 활용한 무선 네트워크 최적화 기법 연구

# 기대 결과

본 프로젝트를 통해 트래픽 예측 기반 자원 할당 방식이 기존 Reactive 방식보다 자원 활용 효율성과 시스템 안정성을 향상시킬 수 있는지 검증하고, AI 기반 자원 관리 기술의 적용 가능성을 분석한다.
