#  트래픽 시나리오 데이터셋 생성 보고서

본 문서는 캡스톤 디자인 프로젝트의 트래픽 예측 모델(LSTM) 학습 및 자원 할당 실험을 위해 생성된 데이터셋에 대한 설명이다. 

---

## 1. 데이터 생성 개요
- **데이터 형식:** CSV (Comma-Separated Values)
- **시간 단위:** 1초 (Continuous Time-series)
- **주요 컬럼:** `time_sec`, `target_rps`, `scenario`, `phase`
- **특이사항:** 현실적인 실험 환경 조성을 위해 모든 데이터에 **무작위 노이즈(Random Noise)**를 포함함

---

## 2. 시나리오별 상세 설명

### ① 타임세일 이벤트 (`sale_event_traffic.csv`)
- **설명:** 이커머스 플랫폼의 정기 타임세일 상황을 모사함
- **흐름:** `평상시(80~100 RPS)` → `유입 증가(최대 250 RPS)` → `피크(500~700 RPS)` → `점진적 감소`
- **용도:** 기본적인 예측 기반 자원 할당 성능 테스트

---

## 3. 데이터 생성 규칙 및 노이즈 반영
- **변동성 구현:** `92 → 88 → 95 → 91`과 같이 매 초마다 값이 변화하도록 설정하여 실제 네트워크 트래픽의 불규칙성을 재현함
- **연속성 보장:** 0초부터 최대 1200초까지 중간에 끊김 없는 시계열 데이터를 보장함
- **구간 특징:** 각 `phase`에 맞는 RPS 범위를 유지하면서도 구간 전이 시 자연스러운 증감 곡선을 그리도록 설계함
